# NLP-Text-Classification
NLP Preprocessing &
Text Classification
A complete machine learning pipeline: from raw text to a high-accuracy classifier using TF-IDF, CountVectorizer, Naïve Bayes, Logistic Regression, and Linear SVC.
About the Dataset
The 20 Newsgroups corpus contains ~18,000 newsgroup posts across 20 categories. We use 4 categories to keep training fast while covering distinct domains. Metadata (headers, footers, quotes) is stripped for a harder, more realistic task.
